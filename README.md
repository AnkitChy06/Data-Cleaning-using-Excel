# Sales Data Cleaning (Excel)

## Project Overview

This project demonstrates the data cleaning process performed on a raw sales dataset using Microsoft Excel.

## Data Cleaning Summary

* Removed duplicate records to eliminate redundant data.
* Converted date values from text format to Excel date format.
* Identified and handled missing (null) values.
* Replaced blank values with appropriate statistical values (such as mean, median, or mode, depending on the column).
* Checked the dataset for consistency and prepared it for further analysis.

## Tools Used

* Microsoft Excel 2021

## Output

The cleaned dataset is accurate, consistent, and ready for data analysis and dashboard creation.
